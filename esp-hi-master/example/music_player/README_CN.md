# 音乐播放器示例

本示例基于 ESP-HI 开发板，实现了多首音频文件的本地循环播放。用户可通过单个按键实现播放、暂停和切换下一首的功能。

## 功能简介

- 支持多首音频文件循环播放（需烧录 audio 分区）
- 单击按键：
  - 空闲时：播放当前音频
  - 播放时：暂停播放
  - 暂停时：切换到下一首并播放
- 音频播放完成后自动暂停
- 音频参数：采样率 44100Hz，单声道，16bit
- 音量最大

## 硬件要求

- ESP-HI 开发板

## 使用方法

1. 准备音频文件，并通过工具生成 audio 分区镜像，烧录到设备指定分区
2. 连接 ESP-HI 开发板至电脑
3. 按如下步骤编译并烧录示例：
   ```bash
   idf.py build
   idf.py -p (PORT) flash monitor
   ```
4. 按下 BOOT 按键即可控制播放/暂停/切歌


## 相关文件

- main/main.c：主程序入口及按键、音频播放逻辑
- audio/：存放音频分区相关文件

如需自定义音频或功能，请参考源码及注释进行修改。
